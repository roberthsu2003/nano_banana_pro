# 資訊圖表常見的風格
根據您提供的資料，資訊圖表常見的風格（或專業領域中的美學趨勢與設計手法）名稱非常多元。這些風格橫跨了極簡結構、文化美學、以及現代 3D 擬真技術。

以下是資料來源中提及的，適用於資訊圖表或視覺傳播領域的常見風格專有名稱：

## 資訊圖表與視覺設計的常見風格

### 1. 極簡主義 (Minimalism)
| 核心理念 | 特徵與應用 |
| :--- | :--- |
| **少即是多 (Less is More)**<br>強調透過簡化視覺元素傳達清晰訊息。 | • **空間留白**：僅保留必要部分，強調功能性與結構。<br>• **視覺表現**：整潔和諧，線條乾淨，對比明確。<br>• **本質**：設計歷程與文化經驗的高度濃縮。 |

#### 💡 實戰 Prompt 組合範例

> [!TIP]
> **給您的建議**：權重最高的通常是 **"Negative Space" (留白)** 和 **"Clean Lines" (俐落線條)**，這兩個詞最能直接控制 AI 畫面的整潔度。

**室內設計 (Interior Design)**
```text
Interior design of a living room, Minimalism, Less is more philosophy, Negative space, Clean lines, Functional furniture, Neutral color palette, white and light oak wood, soft natural lighting, Airy and clutter-free, high resolution, photorealistic.
```
*(客廳室內設計，極簡主義，少即是多哲學，留白，俐落線條，功能性家具，中性色調，白與淺橡木色，柔和自然光，通透且無雜物，高解析度，照片級真實。)*

**平面設計 (Graphic Design)**
```text
A poster design, Swiss Style, Geometric composition, a single red circle in the center, Vast white space, High contrast, Sans-serif typography, Reductionism, simplistic and elegant, 8k.
```
*(海報設計，瑞士風格，幾何構圖，中央一個紅色圓形，大面積留白，高對比，無襯線字體，簡化主義，簡約優雅，8k。)*



### 2. 新日式風格 (Japanese Design Style)
| 核心理念 | 特徵與應用 |
| :--- | :--- |
| 介於「**禪意留白**」與「**資訊極繁**」之間的光譜。 | • **侘寂 (Wabi-Sabi)**：不完美與自然之美，大地色系、粗糙質感。<br>• **留白 (Ma)**：創造閱讀節奏與呼吸感。<br>• **有序的極繁主義**：高密度資訊、多字體與標籤，象徵詳盡與誠意。 |

### 3. 黏土擬真風格 (Claymorphism)
| 核心理念 | 特徵與應用 |
| :--- | :--- |
| 模仿**黏土或充氣玩具**的質感 (Clay UI)。 | • **視覺特徵**：蓬鬆圓角 (High Corner Radius)，馬卡龍色系。<br>• **3D 效果**：雙重陰影 (內/外) 創造懸浮感。<br>• **心理感受**：友善、安全、遊戲化。 |

### 4. 馬卡龍色系 (Macaron Color Palette)
| 核心理念 | 特徵與應用 |
| :--- | :--- |
| **高明度、低飽和度**的粉彩色調 (Pastels)。 | • **情緒**：喚起甜美、舒適、平靜。<br>• **應用**：緩解焦慮的圖表或教材，常搭配 Claymorphism。 |

### 5. 滾動敘事風格 (Scrollytelling)
| 核心理念 | 特徵與應用 |
| :--- | :--- |
| 結合「**滾動**」與「**敘事**」的互動形式。 | • **互動性**：圖表隨滾動變化，控制閱讀節奏。<br>• **優勢**：降低長篇資訊認知負荷，適合複雜敘事視覺化。 |

## 3D 與視覺化技巧的專有名稱

以下設計名詞常作為風格元素或設計方法，與 3D 趨勢或特定視覺效果有關：

### 1. 等距視角 (Isometric Design)
| 定義 | 應用與特徵 |
| :--- | :--- |
| 30 度角網格系統的 2D 繪製 3D 方法 (無透視收斂)。 | • **應用**：繪製地圖、建築結構、複雜系統流程圖。<br>• **特點**：維持物體比例，不隨距離變形。 |

### 2. 維度字體 (Dimensional Typography)
| 定義 | 應用與特徵 |
| :--- | :--- |
| 將文字本身 **3D 化**，成為有厚度、陰影的實體。 | • **趨勢**：2025 年設計趨勢。<br>• **效果**：增強視覺衝擊力。 |

### 3. 漸層色 (Gradient)
| 定義 | 應用與特徵 |
| :--- | :--- |
| 顏色深淺呈現**階段性的變化**。 | • **功能**：比較資料大小 (如地圖深淺)。<br>• **美學**：讓版面衍生節奏感。 |

### 4. 色彩層疊 (Kasane no Irome)
| 定義 | 應用與特徵 |
| :--- | :--- |
| 源自平安時代十二單衣，透過**半透明圖層疊加**。 | • **效果**：創造第三種色彩或深度的暗示。<br>• **美學**：展現層次與細膩感。 |
