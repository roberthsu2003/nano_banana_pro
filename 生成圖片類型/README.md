# Nano Banana Pro 生成圖片類型

生成圖片類型 劃分了適用於 Nano Banana Pro（即 Gemini 3 Pro Image Preview 模型）的提示詞教學內容，旨在幫助使用者根據不同應用場景尋找對應的提示技巧。內容被分為五大核心類別，從專業用途的 資訊圖表（強調 Grounding 和完美文本渲染）到電子商務所需的 產品與商業資產模擬。其中，一項核心優勢體現在 角色與身份一致性 的維持，允許使用者在不同場景中準確保留人臉特徵。此外，指南也涵蓋了需要輸入圖像來操作的 多模態圖像編輯，以及純文字驅動的 敘事性與藝術創作，最後提供了關於解析度、負面提示詞等 核心技術參數 的實戰建議。

## 1. 專業視覺傳達與資訊圖表 (Professional Communication & Infographics)

本章節專注於 NBP 獨有的 Grounding（基礎事實） 和 完美文本渲染 功能，適用於商業、教育和技術文件製作。

| 子章節 | 核心能力與目標 | 範例類型 |
| :--- | :--- | :--- |
| **I.1 高保真文本集成**<br>(High-Fidelity Text) | 確保複雜句子、長篇標題或多語言文本被準確渲染，並融入圖像風格。 | 標誌設計 (Logo)、海報/封面 (Posters)、標語廣告、多語言在地化翻譯。 |
| **I.2 事實性資訊圖表**<br>(Fact-Grounded Infographics) | 運用 Google Search Grounding，生成基於即時或準確數據的視覺內容，以克服模型知識截止的限制。 | 科學圖表、技術規範圖、即時天氣圖表、教育材料、將文章/PDF 壓縮成白板圖。 |
| **I.3 技術與工程藍圖**<br>(Technical & Engineering Blueprints) | 要求模型以精確的幾何和結構細節進行渲染，通常需要特定的專業術語。 | 正交藍圖 (Orthographic Blueprint)、建築設計渲染圖。 |



## 2. 產品與商業資產模擬 (Product & Commercial Assets)

| 子章節 | 核心能力與目標 | 範例類型 |
| :--- | :--- | :--- |
| **II.1 專業產品攝影**<br>(Studio Product Shots) | 使用專業攝影術語（如：三點照明 (three-point lighting)、景深、角度）來生成無瑕疵的商業圖像。 | 化妝品瓶、咖啡杯、電子產品、珠寶的靜態攝影。 |
| **II.2 品牌套件與行銷模擬**<br>(Brand Kits & Mockups) | 維持一致的風格和配色方案，並將品牌標誌自然地整合到多個產品表面上。 | 應用 UI/UX 設計模擬圖、咖啡包裝、隨行杯、廣告看板等一整套品牌視覺資產。 |
| **II.3 草圖/設計稿轉實景**<br>(Sketch to Photorealism) | 將低保真度的輸入（如手繪草圖或平面設計圖）轉換為高逼真度、環境一致的實景照片。 | 將房屋藍圖轉換為完成後的房屋渲染圖、將產品概念草圖變成展間照片。 |

## 3. 角色與身份一致性 (Character & Identity Consistency)

| 子章節 | 核心能力與目標 | 範例類型 |
| :--- | :--- | :--- |
| **III.1 角色表單與多視角**<br>(Character Sheets & Multi-View) | 生成角色不同角度（前、側、後）的圖像，以用於 3D 建模或持續內容創作。 | 遊戲資產的角色轉身圖、多視角肖像照。 |
| **III.2 跨場景一致性**<br>(Cross-Scene Consistency) | 確保角色在不同的動作、服裝或環境光線下，仍能保持其臉部和身體特徵不變。 | 系列漫畫或教學內容中的主角、將現實人物置入虛構景觀。 |
| **III.3 肖像編輯與保真度**<br>(Portrait Editing & Fidelity) | 對人物進行細微的編輯（如更換服飾、妝容或髮型），同時確保臉部特徵完全不受影響。 | 將 logo 添加到 T 恤上並模擬布料紋理、改變髮型。 |

## 4. 多模態圖像編輯與合成 (Multi-Modal Image Editing & Compositing)

| 子章節 | 核心能力與目標 | 範例類型 |
| :--- | :--- | :--- |
| **IV.1 局部替換與 Inpainting**<br>(Inpainting & Replacement) | 僅改變圖像中的特定元素，並確保替換後的物體與原圖的光線、陰影和視角完美融合。 | 改變客廳沙發的顏色/款式、移除背景中的物體。 |
| **IV.2 圖像混合與重構**<br>(Image Blending & Recomposition) | 結合來自 多個圖像輸入 的元素（最多 14 張參考圖）來創造新場景。 | 將一件衣服 (圖 1) 穿在模特兒 (圖 2) 身上，並設定新的背景、複雜的數碼拼貼。 |
| **IV.3 風格轉移與再著色**<br>(Style Transfer & Recoloring) | 將圖像內容轉換為特定的藝術媒介或風格，或精確地控制色彩配置。 | 將照片轉為梵谷的風格、將黑白漫畫/照片轉為彩色。 |
| **IV.4 環境與光線替換**<br>(Environment & Lighting Swap) | 在保持主體不變的情況下，改變場景的整體光線、天氣或時間。 | 將白天照片轉為夜晚、將晴天轉為多霧或下雨。 |

## 5. 敘事性與藝術創作 (Narrative & Artistic Creation)

| 子章節 | 核心能力與目標 | 範例類型 |
| :--- | :--- | :--- |
| **V.1 電影與寫實場景**<br>(Cinematic & Photorealistic Scenes) | 運用攝影術語（如：f/1.8 淺景深、鏡頭類型、金色時光）來控制高逼真度場景。 | 寫實肖像、無人機景觀鏡頭、產品情境照。 |
| **V.2 風格化插畫與貼紙**<br>(Stylized Illustration & Stickers) | 專注於定義藝術媒介、線條風格和著色，創造非寫實的視覺素材。 | 卡通風格貼紙 (Kawaii-style sticker)、水彩畫、概念藝術 (Concept Art)。 |
| **V.3 視覺敘事與分鏡圖**<br>(Visual Storytelling & Storyboards) | 製作用於規劃漫畫或電影分鏡圖的連續視覺序列。 | 漫畫面板 (Comic Panel)、6-8 格的電影分鏡序列圖（需注意連貫性仍是挑戰）。 |
| **V.4 抽象與極簡設計**<br>(Abstract & Minimalist Design) | 創建具有大量留白（Negative Space）的設計，適用於網頁背景或需要疊加文字的素材。 | 極簡主義構圖、抽象 3D 形狀。 |

## 6. 核心技術參數與程式碼實戰 (Technical Parameters & Code Implementation)
| 子章節 | 核心能力與目標 |
| :--- | :--- |
| **VI.1 解析度與長寬比控制** | 如何設定 1K, 2K, 4K 解析度；設定各種長寬比（例如：16:9, 9:16, 5:4）的範例與代碼。 |
| **VI.2 負面提示詞與錯誤緩解** | 針對不同類型的圖像（例如：肖像、資訊圖表）使用最有效的語義負面提示詞，以避免常見瑕疵。 |
| **VI.3 API 工作流程與工具** | 使用 API（例如 gemini-3-pro-image-preview）程式化生成上述圖像類型的代碼範例。 |
