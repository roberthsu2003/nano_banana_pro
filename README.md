# Nano Banana Pro 提示調教學研究

根據最新的資訊（2025年11月發布），**「Nano Banana Pro」** 是 Google 最新的圖像生成模型代號（正式名稱為 **Gemini 3 Pro Image** 的趣味代號）。

與之前的模型不同，Nano Banana Pro 最大的突破在於它具備 **「邏輯思考能力 (Thinking)」** 和 **「精準的文字渲染能力」**。這意味著它不再只是「畫畫」，而是真的「懂」複雜的結構和資訊，因此它非常擅長繪製各種類型的**圖解 (Diagrams)**。
---

### 如何啟用這些功能？

要讓 Nano Banana Pro 畫出上述複雜圖解，請務必在你的工具（如 Gemini Advanced 或 AI Studio）中設定：

1.  **模型選擇：** 選擇 **Nano Banana Pro** (或顯示為 **Gemini 3.0 Pro Image**)。
2.  **模式選擇：** 務必開啟 **"Thinking" (思考型)** 模式。這個模式會讓 AI 先「規劃」圖表的佈局和文字內容，然後再開始畫圖，這樣文字才不會出錯。
3.  **語言：** 可以直接使用**繁體中文**輸入提示詞，它對繁中的支援是目前業界最強的。

### 如何生成優質的圖片?
1. **從簡單的公式開始:**

試試「<製作/打造> <主體> <在場景> <動作> 的圖片」這個公式，然後再逐步調整。例如：「製作貓咪在窗台的陽光下打盹的圖片。」

2. **盡可能詳細說明，發揮你的想像力**

提示應包含你能想到的所有細節，因此比起「生成一張穿紅洋裝的女性圖片」，「生成一張穿紅洋裝的年輕女性在公園跑步的圖片」會是更好的提示。提供的細節越多，Gemini 就越能按指示生成圖片。

3. **請將構圖、風格和圖片品質納入考量**

思考圖片中各元素的排列方式 (構圖)、想呈現的視覺風格 (風格)、期望的圖片品質 (圖片品質) 和顯示比例 (尺寸)。例如：「生成一張模糊的尖刺豪豬在太空中飛行的圖片，風格為油畫，顯示比例為 2:3。」

4. **創意是你的好朋友**

Gemini 非常擅長生成超現實的物件和獨特場景。讓想像力自由馳騁，並試試一些意想不到的組合。

5. **如果不喜歡**

請 Gemini 修改即可。有了圖像編輯模型，只要告訴 Gemini 變更背景、替換物件或新增元素，然後要保留哪些喜愛的細節，就能精準掌控圖像。
